# CourseReco-engine

recommender engine for the Course Recommender project 

## Installation

just clone this repo

```bash
git clone https://github.com/HackerHub-ucdavis/CourseReco-engine
```

## Run

```bash
python start_engine.py
```

## CLI arguments

```bash
python start_engine.py -h
```
